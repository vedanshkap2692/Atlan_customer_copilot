Do not add policies in bulk

Be careful to only add policies one-by-one to a persona. While the SDKs will allow you to add them in bulk, currently this results in a persona where only the final policy in the batch is active at the end of the operation.

API token must be a connection admin

To manage policies for a connection, the API token must be a connection admin on that connection. When you create a connection using an API token, the API token isautomaticallymade a connection admin; however, for any other connection you mustcarry out extra steps to make the API token a connection admin.
